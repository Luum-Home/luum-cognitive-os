# Competitive Arena

> This functionality is implemented as the `/arena` skill. Run `/arena` to use it.
>
> For the full procedure, see `skills/arena/SKILL.md`.

## Overview

The Competitive Arena benchmarks Cognitive OS against 10+ AI coding tools across standardized tasks, measuring quality, speed, cost, and completeness. It runs a suite of tasks on multiple tools, collects structured results, and produces a comparative report that identifies strengths, weaknesses, and improvement opportunities.
