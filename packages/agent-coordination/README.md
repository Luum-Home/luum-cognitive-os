# @luum/agent-coordination

Agent coordination rules — communication bus, sidecars, and per-agent customization overrides

## Install

```bash
cos install @luum/agent-coordination
```

## Components

- `rules/agent-communication.md` (rule) -- Agent Communication Bus Protocol via Valkey pub/sub
- `rules/agent-sidecars.md` (rule) -- Per-agent persistent sidecar memory via Engram
- `rules/agent-customization.md` (rule) -- Per-agent behavioral overrides via customization YAML files
