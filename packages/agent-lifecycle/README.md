# @luum/agent-lifecycle

Agent lifecycle hooks — prelaunch registration and post-completion checkpointing

## Install

```bash
cos install @luum/agent-lifecycle
```

## Components

- `hooks/agent-prelaunch.sh` (hook) -- Register agent tasks in active-tasks.json before launch
- `hooks/agent-checkpoint.sh` (hook) -- Update task status on agent completion
