# @luum/consequence-system

OKR-driven feedback loop — consequence evaluation, trust score validation, and auto-skill generation

## Install

```bash
cos install @luum/consequence-system
```

## Components

- `hooks/consequence-evaluator.sh` (hook) -- Evaluate agent trust score streaks for promote/degrade/disable
- `hooks/trust-score-validator.sh` (hook) -- Validate Trust Report presence and extract scores
- `hooks/auto-skill-generator.sh` (hook) -- Auto-generate reusable skills from complex agent completions
