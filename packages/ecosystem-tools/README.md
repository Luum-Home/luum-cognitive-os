# @luum/ecosystem-tools

External tool integrations for Cognitive OS — agnix, parry, hcom, repomix, Trail of Bits, Context7

## Install

```bash
cos install @luum/ecosystem-tools
```

## Components

- `rules/ecosystem-tools.md` (rule) -- Overview of all integrated external tools
- `rules/parry-integration.md` (rule) -- Parry prompt injection scanner configuration
- `rules/hcom-integration.md` (rule) -- Hcom cross-terminal communication configuration
- `rules/repomix-integration.md` (rule) -- Repomix repo context packing integration
- `rules/trailofbits-skills.md` (rule) -- Trail of Bits security skills documentation
- `rules/context7-auto-trigger.md` (rule) -- Context7 library doc auto-lookup rule
- `hooks/agnix-lint.sh` (hook) -- Config file linter hook (342 rules)
