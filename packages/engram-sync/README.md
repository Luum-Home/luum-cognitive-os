# @luum/engram-sync

Engram memory synchronization hooks — auto-sync on stop, auto-import on start, memu bridge

## Install

```bash
cos install @luum/engram-sync
```

## Components

- `hooks/engram-auto-sync.sh` (hook) -- Auto-sync engram observations on session stop
- `hooks/engram-auto-import.sh` (hook) -- Auto-import engram observations on session start
- `hooks/memu-sync.sh` (hook) -- Sync memory to memu service after tool use
