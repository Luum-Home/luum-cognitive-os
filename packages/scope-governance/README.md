# @luum/scope-governance

Scope governance — proportionality checks and scope creep detection for agent tasks

## Install

```bash
cos install @luum/scope-governance
```

## Components

- `rules/scope-creep-detection.md` (rule) -- Scope creep detection patterns and prevention
- `hooks/scope-proportionality.sh` (hook) -- Validates change scope is proportional to task type
